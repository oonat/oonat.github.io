# oonat.github.io

Personal academic website of [Onat Özdemir](https://oonat.github.io/) — PhD student at the School of Informatics, University of Edinburgh.

Built with [Jekyll](https://jekyllrb.com/) on top of the [al-folio](https://github.com/alshedivat/al-folio) theme. Auto-deployed to GitHub Pages via `.github/workflows/deploy.yml`.

The standalone CVPR 2026 project page at `/ezpc/` lives under [`ezpc/`](ezpc/) and is served verbatim — Jekyll does not process it.

## Local development

```bash
bundle install
bundle exec jekyll serve --livereload
# http://127.0.0.1:4000
```
